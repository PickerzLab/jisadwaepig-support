# 지사돼 지원 페이지

비트코인을 꾸준히 나눠 사고 싶은데 매번 타이밍이 고민되시나요?  
지사돼는 K-AHR999 기반 적립 지표로 오늘의 적립 환경을 매일 한 문장으로 정리해 드립니다.

## 주요 기능

- **지사돼 점수 (0~100)**: 오늘의 비트코인 적립 환경을 점수 하나로 요약
- **K-AHR999**: 업비트 200일 기하평균가 기반 한국형 장기 가치 지표
- **김치프리미엄**: 국내 실행 가격 참고 지표
- **공포탐욕지수 · 이동평균 · 7일 변동률**: 보조 지표 종합
- **급락 신호**: 7일 변동률 -12% 이하 시 구간별 안내
- **Plus**: 차트, 기록, 급락 상세 분석, 투자 성향 설정

&nbsp;

## 자주 묻는 질문

**점수가 높을 때 한꺼번에 많이 사면 더 좋은가요?**  
아닙니다. 지사돼 점수는 매수 권유가 아니라 오늘의 적립 환경이 얼마나 우호적인지 보는 참고 지표입니다. DCA의 핵심은 점수와 무관하게 정해진 금액을 꾸준히 이어가는 것입니다.

**백테스트 수치는 미래 수익을 보장하나요?**  
아닙니다. 모든 수치는 과거 데이터 기반이며 미래 결과를 보장하지 않습니다. 모든 투자 판단과 손익은 전적으로 본인에게 있습니다.

**Plus는 어떻게 구매하나요?**  
앱 내 설정 화면에서 구매할 수 있습니다. 이미 구매하셨다면 동일 화면의 &#x2018;구매 복원&#x2019; 버튼을 이용해 주세요.
